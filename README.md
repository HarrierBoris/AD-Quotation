# AD-Quotation
Automated quotation/proposal generation tool built for pre-sales workflows — turns CRM client data into branded, client-ready Word documents in seconds instead of manual formatting.
# Overview
This tool replaces a manual proposal-creation process with a fully automated pipeline: pull client and deal data from HubSpot, apply a fixed branded layout (fonts, colors, package tiers), and generate a polished .docx proposal ready to send — cutting proposal turnaround time per quote.
# Features

Automated .docx generation with consistent branded styling (custom fonts, colors, table formatting)
HubSpot CRM integration for pulling client/deal data directly into proposals
Three-tier package structuring (Essential / Signature / Luxury) generated dynamically per client
Centralized layout configuration for easy design updates across all generated documents

# Tech Stack
Python · Flask · python-docx · HubSpot API
